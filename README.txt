Инструкция по запуску и использованию программы:

1. Что нужно для запуска?
Операционная система: Windows / Linux / macOS
Компилятор C++: g++ (установится автоматически или вручную)

2. Как установить и запустить?
 1. Скачайте архив с программой и распакуйте в любую папку
 2. Откройте терминал (Linux/macOS) или Командную строку (Windows).
 3. Перейдите в папку с программой: (cd /путь/к/папке/с/программой)
 4. Соберите программу: make (в терминале)
ЕСЛИ "make" НЕ РАБОТАЕТ, ПОПРОБУЙ ВРУЧНУЮ: (g++ main.cpp product.cpp -o product_db)
 5. Запусти программу: 
    * Если Linux/macOS: ./product_db
    * Если Windows: product_db.exe
    
    
   
3. Как пользоваться программой:
Основные функции:
Команда			Действие
1			Показать все товары (из файла products.txt)
2			Добавить новый товар (вводите ID, название, цену, количество)
3			Удалить товар по ID
4			Изменить данные товара (по ID)
5			Найти товары по названию (можно часть слова)
6			Найти товары дешевле указанной цены
7			Выйти из программы (данные сохранятся автоматически)


4. Возможная ошибка:
   Если пишет "файл не найден" при сохранении, то запусти программу от имени админа или перемести файл в другую папку
